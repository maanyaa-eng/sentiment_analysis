# sentiment_analysis

This repository contains a simple sentiment analysis project using a dataset of tweets.
It includes:
Tweets.csv — the dataset used for analysis
sentiment_analysis_code.ipynb — the Jupyter Notebook containing all code and visualizations

# Project Overview
The goal of this project is to analyze tweets and understand whether people express positive, negative, or neutral sentiments about airlines.
The notebook walks through:
Loading the dataset
Exploring tweet sentiment
Basic visualizations
Understanding why some tweets are negative

# Files in This Repository
| File                | Description                                    |
| ------------------- | ---------------------------------------------- |
| **sentiment.ipynb** | Jupyter Notebook with all analysis steps       |
| **Tweets.csv**      | Dataset containing tweets and sentiment labels |

# How to Run
Download or clone the repository
Open sentiment_analysis_code.ipynb in Jupyter Notebook or Google Colab
Run the cells to see the output

# Dataset Info
The dataset includes columns such as:
Tweet text
Sentiment (positive, neutral, negative)
Reason for negative sentiment
Airline name
Tweet timestamp
